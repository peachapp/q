# `iframe`

`iframe`也称作嵌入式框架，嵌入式框架和框架网页类似，它可以把一个网页的框架和内容嵌入到现有的网页中。

优点：

- 可以用来处理加载缓慢的内容，比如：广告。

缺点：

- `iframe`会阻塞主页面的`onload`事件。
- `iframe`和主页面共享连接池，而浏览器对相同域的连接有限制，所以会影响页面的并行加载。但是可以通过`JS`动态给`ifame`添加`src`属性值来解决这个问题，同时也可以解决`iframe`会阻塞主页面`onload`事件的问题。
- 会产生很多页面，不易管理。
- 浏览器的后退按钮没有作用。
- 无法被一些搜索引擎识别。
